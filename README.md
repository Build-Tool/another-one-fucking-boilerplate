### Global requirements

* Linux, Mac
* Node.js 4+
* npm
* MongoDB 3.2+
* Redis 3+
* g++ 4.8+ (for hiredis, for deploying)
* pm2 (for production)
* server for static content (for production, Nginx is recommended)

Some of these requirements may be excessive for you.
